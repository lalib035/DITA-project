HTML & CSS Term Glossary

HTML and CSS code is made up of a variety of components. Understanding these terms is crucial to understanding generic containers.

Term Definition Example
Semantically meaningful element An HTML element that gives its contents additional meaning <em> or <nav>
Semantically meaningless element/generic container An HTML element that groups its contents without adding additional meaning <span>, <div>
In-line element An HTML element that groups a small amount of content within a line of HTML <span> or <em>
Block-level element An HTML element that groups larger amounts of HTML code; often this code would be surrounded by line breaks <div> or <article>
HTML tag The HTML code surrounded in brackets (< and >); can be an opening tag (< >) or a closing tag (</ >)

Opening tag: <span>
Closing tag: </span>
HTML element The opening and closing HTML tags and their contents <span>blue</span>
CSS selector CSS that identifies (or "selects") what section of an HTML document to format

p {
   color: red;}
CSS property CSS that defines what kind of formatting to apply to the selected section

p {
   color: red;}
CSS value CSS that states the specific result of this formatting

p {
   color: red;}
