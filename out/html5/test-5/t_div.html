Using the Generic Container DIV

If you want to contain "block-level" text, you should use the block-level generic container DIV.

In order to use <span>, you will need the following:
  • Text editor
  • HTML document
  • Selection from document that you would like to "contain"

You should also:

  • Confirm that you have selected the appropriate generic container
  • Confirm that there is not a semantic element that would better fit your purpose
  • Determine how you want to identify your selection (“class" or “id" HTML attribute)
The “class” HTML attribute is used when you would like to apply the same grouping to multiple separate sections of the page. The “id” HTML attribute is for selecting unique content and should only be used once per page. Attributes can be added to the opening tag of an HTML element.
  1. Insert the opening <div> tag (<div>) on a line before the first character of the area you want to select. 
    <div>
    Once upon a time… Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ullamcorper facilisis risus, et malesuada magna mollis nec. Quisque varius sem odio, in fermentum eros varius non. Pellentesque eu lectus tempus odio porta suscipit. Vestibulum consectetur accumsan condimentum.
  2. Insert the closing <div> tag (</div>) on a line after the last character of the area you want to select. 
            <div>
	Once upon a time… Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ullamcorper facilisis risus, et malesuada magna mollis nec. Quisque varius sem odio, in fermentum eros varius non. Pellentesque eu lectus tempus odio porta suscipit. Vestibulum consectetur accumsan condimentum.
</div>
  3. Add a class or id attribute to the opening tag. 
    <div id="">
	Once upon a time… Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ullamcorper facilisis risus, et malesuada magna mollis nec. Quisque varius sem odio, in fermentum eros varius non. Pellentesque eu lectus tempus odio porta suscipit. Vestibulum consectetur accumsan condimentum.
</div>
  4. Choose and insert a selector title for your attribute. 
    <div id="first-paragraph">
	Once upon a time… Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ullamcorper facilisis risus, et malesuada magna mollis nec. Quisque varius sem odio, in fermentum eros varius non. Pellentesque eu lectus tempus odio porta suscipit. Vestibulum consectetur accumsan condimentum.
</div>
