Choosing your Generic Container

HTML has two types of generic containers: an in-line generic container (<span>) and a block-level generic container (<div>). The generic container you choose will depend on the type of information you want to contain.

In order to choose your generic container, you will need the following:
  • Text editor
  • HTML document
  • Selection from document that you would like to "contain"
Generic containers allow you to select sections of content (often text). You should only use generic containers if there is no applicable semantic element. Unless otherwise defined in a CSS stylesheet, <span> and <div> don't change the visual presentation of their selection.
