HTML and CSS

Websites are made up of files. Most of these files are written in HyperText Markup Language (HTML), which describes the file's structure, or Cascading Style Sheets (CSS), which describes the file's presentation.

Today, HTML and CSS are used together to separate presentation from content.

HTML is a markup language, which means that code written in HTML describes (or marks) its contents. This description can say something about the code's contents, which may then change how its contents are displayed.

For example, a web writer may want to emphasize a term when it is first used. The writer can use HTML code to mark this term as emphasized. HTML code is written inside brackets, so the HTML "tag," or code, that means emphasize is <em>. The <em> tag means "start emphasis", and </em> tag means "end emphasis." In a sentence, the emphasized term would look like <em>this</em>. The opening and closing tags plus their contents make up an HTML element. Most browsers present the <em> element as italicized text.

CSS is a style sheet language. Style sheet languages can apply formatting to documents written in HTML and other markup languages. Most web writers today use a CSS style sheet document to apply formatting to a separate HTML document or documents.

Styling HTML using CSS requires three things: a selector, a property, and and a value. Selectors identify what part of the HTML document to style (for example, anything in an <em> element or any content in a generic container given the attribute <p class="first-paragraph">). Values state the explicit effects of the CSS, like purple color, dotted border, and Times New Roman font.

Using CSS and HTML, you can produce websites like the ones you view every day.
